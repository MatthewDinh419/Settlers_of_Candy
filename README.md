# Settlers of Candy

Based off the game "Settlers of Catan" that is candy themed along with new features. 


## How to Run:
    1. QTCREATOR 
    2. OPEN FILE/PROJECT 
    3. SETTLERS_OF_CANDY.PRO
    4. BUILD
    5. RUN
